Services

Objekter registrert i moduler

  • Sigletons - det finnes bare én av hver
  • Her hører “business-logikk” hjemme
  • Eksterne tjenester og kode (f. eks. $http) 
myModule.factory('counterService', function () {

    var counter = 0;

    return {
        getCount: function () {
            return counter;
        },

        increment: function () {
            counter++;
        }
    };

});

Eksempler:

  • userService
    • get
    • query
    • getSorted
  • $http (en funksjon)
  • $window (wrapper for window)

Det finnes flere måter å registrere services på! Kan også bruke module.service: 

myModule.service('counterService', function () {

    var counter = 0;

    this.getCount = function () {
        return counter;
    };

    this.increment = function () {
        counter++;
    };

});

Eventuelt module.provider eller module.value
