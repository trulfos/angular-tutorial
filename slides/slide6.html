Factories

Funksjon som produserer verdi

  • Injiseres med avhengigheter
  • Lager et scope for private variabler

Enkelt eksempel uten avhengigheter:

myModule.factory('hello', function () {
    return 'Hello';
});

Med avhengigheter:

myModule.factory('myValue', function (hello, world) {
    return hello + ' ' + world;
});

Utnyttelse av scopet:

myModule.factory('hello', function () {
    var counter = 0;

    return function () {
        return counter++;
    };
});
