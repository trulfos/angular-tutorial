Eksempel: Directive

Factory-funksjon som returnerer et DDO (directive definition object)

myModule.directive('myDirective', function () {

return {
    restrict: 'A',
    link: function (scope, element, attributes) {
        console.log(element);
    }
};

});

HTML-kode der direktivet blir brukt:

<div my-directive id=”myDiv“></div>

OBS: Angular normaliserer navn!
