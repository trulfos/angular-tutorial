Controller

Broen mellom services og scopet

  • Hold logikk i services
  • Funksjoner som sendes videre til f. eks. ng-click

Eksempel:

angular.controller('MyController', function ($scope, userService) {
    
    this.saveUser = function () {
        userService.save($scope.user);
    };

    this.replaceUser = function () {
        userService.replace($scope.replaceUserId, $scope.user);
    };


    this.deleteUser = function () {
        userService.remove($scope.deleteUserId);
    };

});
